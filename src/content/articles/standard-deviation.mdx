Standard deviation describes how far values in a dataset typically sit from the mean.
Small standard deviations mean the values are tightly clustered; larger standard
deviations mean the values are more spread out.

<div class="callout">
  <div class="lab">Definition</div>
  <p>Standard deviation is the typical distance between each value in a dataset and the dataset mean.</p>
</div>

<div class="formula-block">
  s = sqrt( sum((x_i - xbar)^2) / (n - 1) )
  <span class="cap">Sample standard deviation divides by n - 1.</span>
</div>

## How to read the result

For a dataset like `2, 4, 4, 4, 5, 5, 7, 9`, the standard deviation gives a single
spread measure in the original units. That makes it easier to compare variability
between similar datasets.
