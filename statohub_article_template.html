How to calculate standard deviation

What standard deviation measures, the two formulas you'll meet, and a worked example — plus a calculator to run your own data.

Beginner 6 min read Updated Jun 2026

[Intro paragraph — set up the question a reader arrives with: "I have a set of numbers and I want to know how spread out they are." One or two sentences, no notation yet.]

What is standard deviation?

[Plain-English definition. Explain spread around the mean using a quick intuitive contrast — two datasets with the same mean but different spread.]

Definition

Standard deviation is the typical distance between each value in a dataset and the dataset's mean. A low value means the points cluster near the mean; a high value means they're spread out.

The formula

[Explain there are two versions and when to use each — population vs sample — in one short paragraph before showing the math.]

σ = √( Σ(xᵢ − μ)² / N ) Population standard deviation · divide by N
s = √( Σ(xᵢ − x̄)² / (n − 1) ) Sample standard deviation · divide by n − 1 (Bessel's correction)

Step by step

[Walk through the procedure as a short list. Keep each step to one line.]

  • Find the mean of all the values.
  • Subtract the mean from each value and square the result.
  • Add up all the squared differences.
  • Divide by N (population) or n − 1 (sample) to get the variance.
  • Take the square root — that's the standard deviation.
