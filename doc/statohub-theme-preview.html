Frequency Tables: How to Build & Read Them

15 min read 1 calculator Updated Jun 2026

A frequency table turns a raw list of numbers into an organised summary showing exactly how often each value appears — the simplest, most foundational move in descriptive statistics.

Frequency tables are among the first tools you reach for when exploring a new dataset. They underpin the frequency distribution, the relative frequency table, and visual displays like the histogram and frequency polygon. Learning to build and read one well is a foundational skill.

What is a frequency table?

A frequency table lists every distinct value in a dataset alongside the count of how many times that value occurs. The frequency is simply the count — a whole number telling you how many observations fell at that value or in that category.

DefinitionA frequency is the number of times a value of the data occurs. A frequency table collects all those counts into one structure so you can compare them side by side.

Build one, step by step

Take the quiz scores of 20 students. Identify each distinct value, tally the occurrences, then record the counts. Always close with a totals row — the sum must equal n = 20, which is your built-in check.

How to read it

The value with the highest frequency is the mode. The point where cumulative relative frequency crosses 0.50 marks the median. The shape — symmetric, skewed, bimodal — tells you which summary statistics to trust. A finished table is a small map of the whole dataset.

How this calculator works

It scans your list, collects every distinct value, and counts how many times each appears. The relative frequency divides each count by the total n; the cumulative column adds the counts from smallest to largest. The totals must come to n and 1.00 — the calculator's own built-in check.

This is the same procedure you'd do by hand, which is exactly why the full method sits in the guide beside it.
